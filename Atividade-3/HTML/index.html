Lista de Exercício 3 - Desenvolvimento Web Básico

Está página é um exercício de desenvolvimento web básico.

Missão : Aprender e aplicar com consistência os fundamentos de HTML, CSS e JavaScript para criar interfaces web funcionais, acessíveis e visualmente atrativas. Comprometer-me com a prática constante, resolução de problemas e desenvolvimento de projetos reais para evoluir como desenvolvedor front-end.

Visão : Tornar-me um profissional reconhecido por criar soluções criativas e bem estruturadas para a web, contribuindo para projetos que impactem positivamente pessoas e negócios. Almejo dominar as tecnologias do ecossistema front-end, estar em constante atualização e, futuramente, compartilhar meu conhecimento com outros iniciantes
